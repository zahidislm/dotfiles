# inspiration?
de-preztofied core prezto modules, customized for Intel Clear Linux.

# install?
Uses GNU Stow.

# tl;dr
navigate to your home directory

`cd ~`

clone the repo:

`git clone https://github.com/zahidislm/dotfiles`

enter the dotfiles directory

`cd dotfiles`

install the config folders

`stow git zsh`
